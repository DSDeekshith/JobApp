# JobApp Practice project

A Spring Boot MVC project for managing job applications. This system allows users to add and view Alljobs applications, offering a basic platform for tracking job postings and applicant details.

## Features

- Add job
- View a list of all job applications
- Basic form validation

## Technologies Used

- **Backend**: Java, Spring Boot, Spring MVC
- **Frontend**: Thymeleaf templates
- **Build Tool**: Maven
- **IDE**: IntelliJ IDEA / Eclipse / VS Code

## Prerequisites

- JDK 11 or higher
- Maven 3.6+
- An IDE such as IntelliJ IDEA, Eclipse, or VS Code
